# Cash Cow I: The Cashening (solid-garbanzo)

## Requirements
- Python 3.10+
- PyGame 2.5.0+

## Run
```$ python3 src/CowGame/main.py```

## Synopsis
Corey the Cow needs cash. He needs to defeat the Wingdings to take their cash.

## Gameplay
WASD or Arrow Keys to move. Spacebar to attack. Escape to escape.

## Planned Features
- Upgrade Menu
- Power-ups
- Enemy Waves/Stages
- UI Enhancements
- Sound Effects
- Background Music
